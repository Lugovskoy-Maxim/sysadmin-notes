# Sysadmin Notes

Локальный Next.js-блокнот для админских заметок: логины, пароли, ссылки, инструкции и картинки-вложения.

## Возможности

- мастер-пароль при первом запуске;
- шифрование данных в браузере через Web Crypto AES-GCM + PBKDF2;
- хранение заметок в `localStorage`;
- категории, поиск и теги;
- поля для URL, логина и пароля с копированием;
- текст инструкций и добавление изображений;
- адаптивный интерфейс для десктопа и мобильной ширины.

## Запуск

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

## Проверка

```bash
npm run lint
npm run build
```

## Важно

Мастер-пароль не восстанавливается. Если забыть его, расшифровать сейф не получится. Для важных данных делайте резервную копию браузерного профиля или экспортируйте данные отдельным способом перед очисткой браузера.
